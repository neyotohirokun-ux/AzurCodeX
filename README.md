A React TypeScript Webb App

Azur Code X or Azur Codex, is an Azur wiki app that is a tone down version of Azur Lane Wiki. To cater for flexibility when adding a somewhat complex features such as the Live2d, Spine Rendering, recreation of some in game mechanics, etc.

With considerations creating this app to limit the data to be presented for legal rights of the original stakeholders: Manjuu, Yongshi and Yostar.

The said app is a prototype and a fan made. Redistributions is not allowed.
